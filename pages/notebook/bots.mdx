# Bots

Some info on botmaking and stuff.

## Let's make a (very simple) Mastodon bot

What do we need?

1. A Masdodon account https://bne.social
2. A Deno Deploy account https://dash.deno.com
3. Create a Mastodon "App" in Settings https://bne.social/settings/applications
4. Grab your App access token
5. A JavaScript Mastodon library https://esm.sh/mastodon
6. Your instance's API enpoint https://bne.social/api/v1/

This code toots a toot:

```javascript
const masto = new Masto({
  access_token: "access_token_here",
  api_url: "https://bne.social/api/v1/",
});

const response = await masto.post("statuses", {
  status: "Hello this is just a test!",
});

```
